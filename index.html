🔥 إعداد قاعدة البيانات السحابية

لمزامنة البيانات بين جميع الأجهزة، تحتاج إلى ربط الموقع بـ Firebase (مجاني تماماً). اتبع الخطوات أدناه مرة واحدة فقط.

1
إنشاء مشروع Firebase

اذهب إلى console.firebase.google.com وسجّل دخول بحساب Google، ثم اضغط "Add project" وأدخل أي اسم للمشروع.

2
تفعيل Firestore Database

من القائمة الجانبية اضغط Build → Firestore Database → Create database، اختر "Start in test mode" ثم اختر أقرب منطقة لك واضغط Enable.

3
إضافة تطبيق ويب

من إعدادات المشروع (أيقونة التروس) → "Your apps" → اضغط </> (Web)، أدخل أي اسم واضغط Register App.

4
نسخ إعدادات التطبيق

ستظهر لك كتلة كود فيها firebaseConfig. انسخ النص الذي يشبه هذا الشكل وضعه في الخانة أدناه:

{ apiKey: "...", authDomain: "...", projectId: "...", ... }

⚠️ تعارض في مواعيد الحلاق

💡 كل حلاقة تستغرق حوالي 30 دقيقة. يُنصح باختيار وقت مختلف لتجنب التعارض.
